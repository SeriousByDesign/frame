# LinkedIn Profile Creator
Version: 1.0
Domain: LinkedIn profile authorship — building a complete, professional LinkedIn profile from an existing CV

Use this cartridge to create a compelling LinkedIn profile from scratch. You bring your CV; the cartridge guides you through extracting the right material, planning each section's angle, and writing publish-ready copy — section by section.

---

## Phases

| # | Phase | Role | Goal |
|---|---|---|---|
| 1 | **SHAPE** | CV Analyst | Extract CV material, surface additional experience, establish positioning and tone |
| 2 | **DESIGN** | Profile Strategist | Plan each LinkedIn section — what to include, what angle to take |
| 2.5 | **BREAKDOWN** | Orchestrator | Enumerate profile sections as build units |
| 3 | **BUILD** | Profile Writer | Write each section as polished, publish-ready copy |
| 4 | **CHECK** | Profile Reviewer | Verify accuracy, tone consistency, and readiness to publish |

---

## Unit sizing hint

Each LinkedIn profile section is one unit: Headline, About, each Experience entry, Education, Skills, and any optional sections (Certifications, Projects, Volunteering). The number of units varies by CV length.

---

## Draft output

BUILD writes profile sections to `.draft/linkedin-profile-creator/` at the project root. Each section is a separate file.
